# UI 美化设计文档 — 极简风格

> 版本：V1.0 | 状态：ACTIVE | 日期：2026-05-27

---

## 一、现状

当前风格：深色背景 (#1a1a2e) + 红色点缀 (#e94560)，暗黑电竞风。

## 二、目标风格

**极简 / 性冷淡 / MUJI 风**

关键词：留白、克制、干净、无装饰

## 三、配色方案

```css
:root {
  /* 背景 — 暖白系 */
  --bg:              #fafaf8;       /* 主背景 */
  --bg-card:         #ffffff;       /* 卡片/面板 */
  --bg-hover:        #f0f0ed;       /* 悬停 */
  --bg-input:        #f5f5f3;       /* 输入框 */

  /* 文字 — 灰黑系 */
  --text-primary:    #1a1a1a;       /* 主文字 */
  --text-secondary:  #8c8c8c;       /* 次要文字 */
  --text-tertiary:   #b0b0b0;       /* 占位符/提示 */

  /* 边框/分割 — 极淡 */
  --border:          #e8e8e4;       /* 边框 */
  --divider:         #f0f0ed;       /* 分割线 */

  /* 强调 — 克制 */
  --accent:          #2c2c2c;       /* 按钮/链接（深灰，不抢眼） */
  --accent-hover:    #000000;
  --accent-soft:     #e8e8e4;       /* 微妙强调 */

  /* 状态 */
  --success:         #7d9f7a;       /* 成功绿（低饱和） */
  --warning:         #c9a96e;       /* 警告黄（低饱和） */
  --danger:          #d4776a;       /* 错误红（低饱和） */

  /* 圆角 — 统一 */
  --radius-sm:       4px;
  --radius-md:       8px;
  --radius-lg:       12px;

  /* 阴影 — 极轻 */
  --shadow-sm:       0 1px 2px rgba(0,0,0,0.04);
  --shadow-md:       0 2px 8px rgba(0,0,0,0.06);

  /* 字体 */
  --font:            "Inter", -apple-system, "SF Pro Display", "Noto Sans SC", sans-serif;
  --font-mono:       "JetBrains Mono", "SF Mono", monospace;
}
```

## 四、全局风格规则

### 4.1 去掉的东西
- ❌ 所有渐变背景
- ❌ 大红大绿的强调色
- ❌ 粗边框和重阴影
- ❌ 过度的 hover 动画

### 4.2 保留/强化
- ✅ 大量留白（padding/gap 15-30% 以上）
- ✅ 细字体（font-weight 400 为主，标题 500）
- ✅ 单色图标（不彩色的 SVG）
- ✅ 微妙过渡（0.15s ease，不带夸张变化）

## 五、各区域设计

### 5.1 侧边栏（Settings）
- 背景：白色卡片浮在主背景上
- 圆角 12px，细阴影
- 标题小字灰色，不用彩色
- 输入框细边框，聚焦时边框变深

### 5.2 标签栏
- 极简 Tab：文字 + 底部 2px 线条指示
- 活跃 Tab 底部线深色，非活跃无色
- 不设背景块、不设圆角

### 5.3 消息气泡
- 用户消息：浅灰背景 `#f0f0ed`，右对齐
- AI 消息：无背景，左对齐
- 去掉彩色角色标签，用极淡的「你」「AI」小字灰色
- Markdown 代码块：浅灰底 `#f5f5f3`，左侧细竖线

### 5.4 输入框
- 底部固定，背景白色
- 细圆角输入框，placeholder 灰色
- 发送按钮：极简方形灰色按钮，不设彩色

### 5.5 文件浏览器
- 目录树：极简展开箭头，文件图标单色 svg
- 查看器：白色面板，代码块同消息区风格

### 5.6 滚动条
- 宽度 4px，极细
- 颜色 `#d0d0d0`，hover 时 `#aaa`

## 六、排版规范

```
标题：18px / 500 weight / #1a1a1a
副标题：14px / 400 weight / #8c8c8c
正文：15px / 400 weight / #1a1a1a
代码：13px / 400 weight / JetBrains Mono
小字：12px / 400 weight / #b0b0b0
行高：1.6（正文）/ 1.4（标题）
段落间距：12px
```

## 十、预览参考

**配色对比：**
```
现 在：深蓝黑底 + 红点缀 + 电竞风  → 暗沉、激烈
改造后：暖白底 + 黑白灰 + MUJI 风   → 干净、克制、留白
```

**关键词：** 不要渐变、不要彩色、不要花哨。Less is more.
