# ac-project-plan — Assistant Web Chat 项目规划

## 一、范围定义

### ✅ 在范围内
- 通过 Web 浏览器访问 AI 助手对话
- 支持 Token 认证（多人可共用）
- 支持选择不同 Agent（assistant/main/dev-cat 等）
- 聊天界面（暗色主题、消息历史、Markdown 渲染）
- 服务器始终保持可访问
- 完整的项目管理和记忆归档体系
- 定期自动健康检查

### ❌ 在范围外
- 用户注册/登录系统（使用共享 Token）
- 多轮对话上下文持久化（当前为单轮 stateless）
- 流式输出（SSE/WebSocket streaming）
- 移动端原生 App
- 文件上传/附件
- 多语言国际化
- 商业化/付费
- 数据加密传输外的额外安全层（如 Cloudflare Access）

## 二、阶段拆解

```
v0.1 MVP 搭建（已完成）
    └── v0.2 稳定性加固（当前）
            ├── v0.3 域名+持久化
            └── v0.4 项目管理系统
```

### 关键路径
```
cloudflared systemd 服务 ──→ 域名绑定 ──→ 生产就绪
                                         ──→ 安全加固
```
