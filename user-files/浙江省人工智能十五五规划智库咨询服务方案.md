浙江省人工智能领域"十五五"规划工作
智库咨询服务方案

（建议稿）

编制说明

2026年5月12日，《重庆市"十五五""人工智能+"发展规划（2026—2030年）（征求意见稿）》全文公布并公开征求意见，成为全国首批面向"十五五"系统谋划"人工智能+"的省级（直辖市）顶层设计之一。该规划以"人工智能+"为主线，构建了"总体要求—算力·数据·算法等核心要素支撑—'人工智能+'重点行业赋能—产业生态与主体培育—空间布局—安全治理—保障措施"的完整逻辑体系，对各省（区、市）谋划本地区人工智能"十五五"规划具有重要参照价值。

本方案以浙江省人民政府（或省发展改革委、省经信厅等牵头部门）拟开展的人工智能领域"十五五"规划编制工作为服务对象，参照重庆规划的框架逻辑与谋划深度，结合浙江省数字经济先发优势、民营经济活跃、"互联网+"基础雄厚的省情特征，提出系统化的智库咨询服务方案。

———————————————————————————————————————————

一、项目背景与服务需求

（一）政策背景

1. 国家层面。党的二十届相关全会和国家"十五五"规划纲要将人工智能作为引领新一轮科技革命和产业变革的战略性技术，"人工智能+"行动持续深化。国家在算力、数据、大模型、应用场景等方面密集出台政策，要求各地因地制宜、错位发展。

2. 区域竞争层面。北京、上海、广东、重庆等地已陆续启动或公布"十五五""人工智能+"专项谋划。重庆征求意见稿的发布，标志着省级人工智能五年规划进入密集编制期，浙江作为数字经济第一方阵省份，亟需高质量、有辨识度的顶层设计以保持先发优势。

3. 浙江自身需求。浙江"十四五"期间数字经济核心产业增加值持续领跑全国，杭州"六小龙"等现象级企业涌现（如深度求索DeepSeek、宇树科技、群核科技、强脑科技等所代表的杭州创新生态），城市大脑、政府数字化转型走在前列。但同时存在高端算力芯片受制、原始算法创新与头部省市仍有差距、行业大模型落地深度不足等短板，需要前瞻谋划予以系统破解。

（二）服务需求

委托方拟通过引入高水平智库力量，为浙江省人工智能领域"十五五"规划的研究、编制、论证、衔接全过程提供专业咨询服务，重点解决：
- 战略定位与发展目标的科学测算与对标；
- 算力、数据、算法、人才等核心要素的支撑路径设计；
- "人工智能+"重点行业赋能的场景遴选与路线图；
- 产业生态、空间布局、安全治理、保障机制的系统设计；
- 与国家规划、省内相关专项规划的衔接协调。

———————————————————————————————————————————

二、服务目标

（一）总体目标
为委托方提供一套逻辑严密、数据扎实、对标精准、措施可行、具有浙江辨识度的人工智能"十五五"规划研究成果与编制服务，支撑形成高质量的规划文本（送审稿）及配套成果。

（二）具体目标
1. 形成1份《浙江省人工智能产业发展现状评估与"十五五"发展形势研究报告》；
2. 形成1套规划核心指标体系及目标值测算方案；
3. 形成1份《浙江省人工智能领域"十五五"发展规划》文本（含框架稿、征求意见稿、送审稿多轮迭代）；
4. 形成若干专题研究成果（算力、数据、行业赋能、安全治理等）；
5. 协助完成规划论证、意见征集、专家咨询、上下衔接等配套工作。

三、服务内容（参照重庆规划框架展开）

本方案在内容设计上对标《重庆市"十五五""人工智能+"发展规划》的章节逻辑，分七大板块开展研究与编制咨询服务。

模块一  总体研究：现状评估、形势研判与战略定位
- 对浙江省"十四五"以来人工智能产业发展进行全面评估（产业规模、企业主体、算力布局、应用场景、创新能力、人才储备）；
- 横向对标北京、上海、广东、江苏、重庆等省市，明确浙江的优势、短板与赛道选择；
- 研判全球与全国人工智能技术演进、产业格局、政策走向；
- 提出浙江人工智能"十五五"的战略定位（建议方向：打造全球人工智能创新策源地之一、全国"人工智能+"应用第一省、世界级智能产业集群），明确指导思想、基本原则、发展目标与核心指标体系（量化到2030年）。

模块二  核心要素支撑：算力、数据、算法
- 算力：研究浙江智能算力基础设施布局（杭州、宁波等智算中心、"东数西算"长三角枢纽节点衔接、算力调度平台），提出算力规模、绿色算力、自主算力比例等目标与建设路径；
- 数据：研究公共数据授权运营、行业高质量数据集建设、数据要素市场（浙江数据交易所）、数据跨境流动等，对接浙江公共数据领域立法先发优势；
- 算法：研究基础大模型与行业大模型协同发展路径，发挥浙江开源生态、头部企业（阿里通义、之江实验室、杭州创新企业群）作用，提出算法攻关与开源开放策略。

模块三  "人工智能+"重点行业赋能
结合浙江产业特色，遴选并设计赋能路线图，重点包括：
- AI+制造（智能工厂、未来工厂、产业大脑，对接浙江"415X"先进制造业集群）；
- AI+消费与电商（直播电商、智能营销、跨境电商，发挥浙江平台经济优势）；
- AI+现代服务业与金融（数字贸易、智慧物流、金融科技）；
- AI+城市与政务治理（城市大脑、"民呼我为"、政府数字化转型升级）；
- AI+医疗健康、教育、文化旅游、农业（数字乡村）、交通、能源、环保等民生与重点领域；
- AI+具身智能/机器人（依托宇树、强脑等先发企业，布局人形机器人与具身智能新赛道）。

模块四  产业生态与主体培育
- 创新平台体系（之江实验室、湖畔实验室、高校院所、新型研发机构）；
- 企业梯队培育（领军企业、专精特新、独角兽、初创企业"雨林式"生态）；
- 创新链产业链资金链人才链深度融合机制；
- 产业基金、风险投资、概念验证与中试平台等。

模块五  空间布局
- 研究"杭州核心引领、宁波温州等多极支撑、全省协同"的空间格局；
- 谋划杭州城西科创大走廊、之江实验室、人工智能产业园区、特色小镇等承载平台；
- 推动山区海岛县差异化、特色化参与。

模块六  安全与治理
- AI安全（数据安全、算法安全、伦理风险、深度伪造治理）；
- 治理体系（分级分类监管、伦理审查、标准规范、可信AI）；
- 衔接国家《生成式人工智能服务管理暂行办法》及浙江地方探索。

模块七  保障措施
- 组织领导、政策支持、财政金融、人才引育、对外开放合作、试点示范、规划实施评估机制等。

四、服务方式与技术路线

（一）技术路线
"调研评估 → 对标研判 → 框架设计 → 专题攻关 → 文本编制 → 论证衔接 → 迭代完善"七步法闭环推进。

（二）主要研究方法
1. 文献研究与政策梳理：系统梳理国家及兄弟省市（重庆、京沪粤苏等）规划文件；
2. 实地调研与座谈：走访重点企业、园区、平台、主管部门（建议覆盖杭州、宁波、温州、嘉兴等地）；
3. 问卷调查与数据采集：面向企业、科研机构开展产业摸底；
4. 定量测算：运用产业增加值、复合增长率、投入产出等模型测算指标目标；
5. 专家咨询与德尔菲法：组织多轮专家论证；
6. 对标分析：与领先省市及国际标杆开展系统对标。

（三）服务方式
驻点服务与远程支持相结合；提供规划起草执笔、专题报告撰写、专家组织、论证会支撑、意见汇总修改、汇报材料制作等全过程嵌入式服务。

五、进度安排（建议6—9个月，可按委托方要求调整）

阶段一  调研评估（第1—2月）
- 启动会、资料收集、实地调研、现状评估报告；
- 交付：《现状评估与形势研究报告》《调研纪要》。

阶段二  框架设计与专题研究（第3—4月）
- 战略定位、目标指标测算、规划框架（大纲）、各专题报告；
- 交付：规划框架稿、指标体系方案、各专题研究报告。

阶段三  文本编制（第5—6月）
- 形成规划征求意见稿；组织内部论证修改；
- 交付：规划征求意见稿。

阶段四  论证衔接与完善（第7—8月）
- 专家论证、部门征求意见、上下衔接、风险评估、合法性审查支撑；
- 交付：专家论证意见汇总、修改说明、规划送审稿。

阶段五  收尾交付（第9月）
- 配合报批、汇报材料、最终成果归档。
- 交付：规划送审稿（最终稿）、全套成果资料。

七、服务团队

建议组建"首席专家 + 规划编制组 + 专题研究组 + 数据测算组 + 项目协调组"的复合型团队：
- 首席专家：人工智能/数字经济领域权威学者或资深规划专家，把控方向；
- 规划编制组：具有省级规划编制经验的政策研究人员（执笔主体）；
- 专题研究组：算力、数据、行业应用、安全治理等领域专家；
- 数据测算组：产业经济与统计分析人员；
- 项目协调组：负责调研组织、会议保障、进度管控。
（注：实际人员配置与履历另附团队介绍及类似业绩。）

———————————————————————————————————————————

八、质量保障与风险控制

1. 三级审核制度：执笔自审—专题组复核—首席专家终审；
2. 多轮论证机制：内部研讨 + 外部专家论证 + 部门征询；
3. 进度管控：节点交付 + 周报 + 月度协调会；
4. 保密管理：签订保密协议，规范涉密数据与资料管理；
5. 衔接保障：设专人负责与国家规划、省内相关专项规划的对接，避免冲突与重复。
