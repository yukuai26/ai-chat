### 1.3 ⚠️ 半通/不稳定

| 源 | 情况 | 原因 |
|----|------|------|
| 微信公众号·单篇文章 | 你发链接→能抓正文 | 文章页公开，但**无法自动订阅**某号最新推送（微信无公开API） |
| 搜狗微信搜索 | 入口200可访问 | 反爬强（验证码+cookie），不能长期自动抓 |
| 路透 Reuters | RSS失效/401 | 官方RSS已停，需第三方中转 |
| Reddit JSON | 走代理能连但返回非JSON | 拦截数据中心IP，返回验证页 |
| Twitter/X | 网页200 | 抓推文需登录态，免费API已封，Nitter镜像大批死 |

### 1.4 ❌ 爬不了（及原因）

| 源 | 为什么 |
|----|--------|
| **微博热搜** | 接口403，必须带登录cookie，反爬严易封 |
| **抖音** | 接口签名加密(X-Bogus/_signature)，需逆向 |
| **小红书** | 接口签名加密+风控严，需登录态+逆向 |
| **公众号订阅式追踪** | 微信无公开API，只能靠搜狗(不稳)或第三方付费RSS |
| **石墨文档** | 登录后JS动态加载+账号鉴权，抓取工具拿不到登录态 |
| **推特原文/特定账号** | 免费API已封，Nitter镜像失效，需付费API |

### 2.2 单只股票能拿多细 ✅（远超买卖五档）

- **实时盘口**：现价/今开/昨收/最高/最低/成交量/成交额 + **买一~买五、卖一~卖五**（含挂单量）+ 时间戳
- **K线历史**：分钟/日/周/月线，前复权/后复权/不复权，可取多年（理论到上市首日）
- **资金流向**：主力净流入（实测茅台今日 -2.72亿），可分超大单/大单/中单/小单
- **估值**：市盈/市净/市值/换手/振幅/52周高低

### 2.5 美股"延迟"说明 + 美国本土 API

- **为什么延迟**：实时逐笔行情归交易所(NYSE/纳斯达克)所有，是付费产品。全球免费源(无论新浪还是Yahoo)通行做法=给15分钟延迟数据。**与"用中国软件"无关，美国本土免费API同理**。
- **美国本土免费API（实测，走代理）**：
  - **Yahoo Finance** ✅ 免费无需key，全球市场，字段最全（含盘前盘后），**首选**
  - **Alpha Vantage** ✅ 拿到IBM全字段，需免费key（demo可用）
  - **Finnhub** ⚠️ 需注册免费key
  - **Stooq** ⚠️ 触发人机验证
- **真·零延迟实时美股 + Level-2** = 全球都要付费（券商/Polygon/IEX付费档）
